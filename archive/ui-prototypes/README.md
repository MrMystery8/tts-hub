# Archived UI Prototypes

These directories preserve interface experiments that are not part of the supported TTS Hub runtime or FYP implementation evidence. The supported clients are `desktop/` and `mobile/` at the repository root.

| Archived location | Historical source | Purpose |
|---|---|---|
| `legacy-react/` | `frontend/` and `new_webui.py` | Earlier React implementation and launcher |
| `react-design/` | `claude_ui/` and `claude_ui.py` | React design exploration |
| `static-reference/` | `claude_original/` and `claude_original.py` | Static design reference |
| `legacy-custom/` | `custom_ui/` | Earlier custom static interface |

The archive was formalized on 2026-07-19. Its contents are retained for development traceability only, are excluded from supported commands and tests, and are not maintained as runnable applications.
